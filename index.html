Base de datos en la nube. Accede desde cualquier dispositivo con tu cuenta de Google.
Cómo obtener la URL:
1. Abre script.google.com → Nuevo proyecto
2. Pega el código del archivo .gs incluido
3. Implementar → Nueva implementación → Aplicación web
4. Ejecutar como: Yo · Quién puede acceder: Cualquiera
5. Copia la URL y pégala aquí arriba
